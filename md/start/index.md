# Начало работы с интерфейсом базы данных

Эту справку всегда можно открыть используя _Меню -> Справка -> Открыть "Начало работы"_

## Основные элементы интерфейса

Интерфейс приложения состоит из панелей, содержащих разделы базы данных и предоставляющие возможности редактирования.
Каждую панель можно скрывать или перемещать по аналогии с окнами. Элементы управления показом панелей находятся в левой панели инструментов.

### Панель вкладок

Основная панель, недоступная для удаления или перемещения, содержит набор вкладок, открываемых из других панелей. Это могут быть как вкладки "Только для чтения" так и редакторы, например Табличные. Основные элменты управления текущей активной вкладкой находятся в верхней панели интрументов. Каждая вкладка может добавлять свои элементы, если ее активировать.

### Верхняя панель инструментов

Верхняя панель инструментов предоставляет элменты управления для текущей активной вкладки.

### Левая панель инструментов

Левая панель инструментов предоставляет элементы управления показом панелей, которые описаны ниже.

### Панель "Объекты"

Панель объектов отражает все элементы базы данных, которые привязаны к местности. Здесь нет информации об измерениях, только горные объекты, скважины и станции.
Управлять элментами можно открыв контекстное меню на нужном элементе. Для добавления нового объекта необходимо использовать контекстное меню его родителя - _Контекстное меню -> "Добавить"_

### Панель "Быстрый просмотр"

Панель "Быстрый просмотр" предоставляет основную информацию о выделеном объекте. Это может быть объект из "Объекты" или любой другой, отражающей объекты БД.

### Панель "Сопутствующие материалы"

Панель "Сопутствующие материалы" предоставляет информацию о файлах для текущего выделеного объекта, то есть работает точно так же как и "Быстрый просмотр".

Если для выделеного обьекта сопутствующие материалы не предусмотрены, на панели будет написано "Недоступны для этого объекта" 

### Панель "Разгрузка"

Панель "Разгрузка" о всех разгрузкочных замеров доступных в базе данных и предоставляет элементы управления для них.

Каждый элемент списка это набор замеров - все измерения с одной скважины. Двойной клик по элементу списка открывает разделы этого набора замеров. Двойной клик по разделу открывает редактор.
