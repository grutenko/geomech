Начало работы с интерфейсом базы данных «Геомеханика»

Этот документ всегда можно открыть, используя Меню → Справка → Открыть
«Начало работы».

1. Панель инструментов

Окно программы содержит две панели инструментов.

  • Первая панель включает набор переключателей для управления отображением различных панелей управления разделами базы данных. Здесь можно выбрать необходимые панели для работы с физико-механическими свойствами, горными ударами и объектами.

  • Вторая панель отвечает за элементы управления для активной вкладки редактора. На ней всегда отображаются общие элементы управления, такие как сохранение, копирование, вставка и отмена действий. Дополнительные элементы появляются в зависимости от того, какая вкладка выбрана. Например, если выбрана вкладка с инструментами редактирования, могут быть доступны специальные команды для работы с содержимым.

2. Меню

Меню программы находится в верхней части окна и состоит из следующих основных разделов:

  • Файл — содержит стандартные команды, такие как открытие, сохранение, экспорт и закрытие базы данных.

  • Правка — позволяет работать с буфером обмена (копирование, вставка), а также отменять или повторять действия.

  • Вид — включает настройки отображения интерфейса, такие как скрытие или отображение панелей и элементов.

  • Параметры — предоставляет доступ к настройкам базы данных и интерфейса.

  • Утилиты — содержит дополнительные инструменты для анализа и работы с базой данных.

  • Справка — позволяет открыть документацию и получить информацию о программе.

3. Вкладки

Под меню находится панель с вкладками, которая позволяет переключаться между различными разделами программы:

  • [Чтение] — вкладка для просмотра и анализа данных.

  • Горные удары — вкладка для работы с данными по горным ударам.

  • Физ. мех. свойства — вкладка для анализа физических и механических свойств материалов.

  • Разгрузка — вкладка для работы с процессами разгрузки и подготовки данных.

4. Сопутствующие материалы и быстрый просмотр

В правом углу интерфейса находятся кнопки:

  • Быстрый просмотр — открывает доступ к ключевым данным и результатам анализа для быстрого ознакомления.

  • Сопутствующие материалы — позволяет быстро загружать и просматривать связанные с базой данных документы, отчеты или файлы.

5. Строка состояния

Внизу окна программы находится строка состояния, которая отображает текущую активность, загруженные данные и подсказки по использованию элементов интерфейса. Здесь можно увидеть, на каком этапе работы вы находитесь и какие процессы происходят в фоновом режиме.
