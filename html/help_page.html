Начало работы с интерфейсом базы данных «Геомеханика»

Этот документ всегда можно открыть используя Меню → «?» → Открыть «Начало работы»

1. Общая информация об интерфейсе

Основное окно интерфейса представляет из себя набор вкладок, которые открываются в основном из раздела связанных данных панели «Объекты» и содержат таблицы для тех или иных измерений.

2. Панель «Объекты»

Все объекты доступные в БД отображаются в панели «Объекты». Панель разделена два дерева: верхнее отображает физические объекты (Горные массивы, Горизонты, Скважины), нижняя панель отображает связанные данные для выбранного объекта в верхнем дереве. Первый элемент нижнего дерева всегда отображает свойства выбранного объекта.

Панель можно скрыть или показать используя: Меню → Вид → [Объекты] показать/скрыть панель или используя сочетание CTRL+ALT+O либо кликнув на соответствующую кнопку на панели слева

Управлять деревом можно используя контекстное меню на объекте или данных.

3. Панель «Сопутствующие материалы»

Панель «Сопутствующие материалы» отражает файлы для выбранного объекта в панели «Объекты»

Панель можно скрыть или показать используя: Меню → Вид → [Сопутствующие материалы] показать/скрыть панель или используя сочетание CTRL+ALT+D либо кликнув на соответствующую кнопку на панели слева

4. Панель «Быстрый просмотр»

Панель «Быстрый просмотр» просто выводит информацию о текущем выбранном объекте.

Панель можно скрыть или показать используя: Меню → Вид → [Быстрый просмотр] показать/скрыть панель или используя сочетание CTRL+ALT+R либо кликнув на соответствующую кнопку на панели слева
